# Salary Expectations For This Account

Copy this file to:

```text
config/accounts/<account>/salary.md
```

Fill it with real salary expectations for that hh.ru account.

Example structure:

- Expected salary: from ... RUB net.
- Minimum acceptable salary: ... RUB net.
- If asked about salary level, answer: "...".
- Can discuss the range depending on tasks, format, and benefits.
