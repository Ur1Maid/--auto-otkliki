# ROADMAP — источник истины для Ralph

> Это рабочий backlog автономного цикла Ralph. Цикл на КАЖДОЙ итерации читает этот файл,
> берёт **одну** верхнюю невыполненную задачу `[ ]` из активного майлстоуна, делает её,
> верифицирует, коммитит и отмечает `[x]`. Держи задачи маленькими (≈1 коммит каждая).
>
> Формат задачи: `- [ ] (ID) Заголовок — Acceptance: <как проверить, что готово>`
> Активный майлстоун помечен **▶ ACTIVE**. Когда все задачи майлстоуна `[x]`, сними метку
> ACTIVE, поставь её на следующий и продолжай. Когда всё `[x]` — выведи `RALPH: ALL DONE`.
>
> **Порядок = приоритет.** Сначала фундамент (M1–M2), затем продуктовые фичи (M3–M7),
> затем общий хардненинг (M8–M10), который страхует фичи.
>
> **Инварианты на весь backlog (проверяет orchestrator/security-reviewer):**
> отклики по 3 резюме одновременно — сохранить; honesty-гардрейлы DeepSeek — не ослаблять;
> ключ/сессии/PII — не логировать/не коммитить; реальный браузерный флоу — не гонять против
> реальных аккаунтов в тестах. DOM-задачи требуют живого HTML от пользователя (research-first):
> их acceptance = чистый парсер на сохранённом санитизированном фикстуре + ручная проверка.
