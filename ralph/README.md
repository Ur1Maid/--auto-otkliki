# Ralph — автономный цикл доработок

Ralph — это техника «бесконечного цикла»: один и тот же промпт скармливается агенту Claude Code
снова и снова. Агент каждый раз без памяти берёт **одну** задачу из бэклога, делает её,
верифицирует, коммитит и отмечает выполненной. Состояние живёт в файлах, а не в контексте модели.

## Как это устроено здесь
- **`PROMPT.md`** — промпт одной итерации. Делает Claude оркестратором (`.claude/agents/orchestrator.md`):
  research-first → реализация (можно делегировать субагенту `implementer`/Sonnet) → верификация
  (`npm test` + `code-reviewer`/`security-reviewer`/Opus) → коммит → отметка `[x]`.
- **`ROADMAP.md`** — единственный источник истины. Бэклог по майлстоунам, активный помечен `▶ ACTIVE`.
- **`PROGRESS.md`** — append-only журнал: что сделано / заблокировано.
- **`ralph.ps1`** / **`ralph.sh`** — сам цикл (Windows / POSIX).
- **`logs/`** — вывод каждой итерации (в git не попадает).

## Предусловия
1. Установлен Claude Code, команда `claude` доступна в PATH.
2. `npm.cmd install` выполнен (нужен `node --test`, Node `>=20`).
3. Заполнен `.env` (если задача требует AI-прогона; для M1 не обязательно — тесты без сети/браузера).
4. Чистое рабочее дерево на ветке `ralph/auto` (скрипт сам создаст/переключит ветку).

## Запуск
```powershell
# Windows (рекомендуемый, безопасный режим: правки авто-применяются, но инструменты ограничены allowlist)
powershell -ExecutionPolicy Bypass -File ralph\ralph.ps1 -MaxIterations 10

# Полная автономия без подтверждений (быстрее, но рискованнее — см. ниже)
powershell -ExecutionPolicy Bypass -File ralph\ralph.ps1 -MaxIterations 10 -Yolo
```
```bash
# POSIX
ralph/ralph.sh -n 10
ralph/ralph.sh -n 10 --yolo
```

## Контроль и остановка
- **Стоп-файл:** создай `ralph/STOP` — цикл завершится после текущей итерации.
  ```powershell
  New-Item -ItemType File ralph\STOP
  ```
- **Лимит:** `-MaxIterations` / `-n` (по умолчанию 30).
- **Авто-стоп:** при `RALPH: ALL DONE` (бэклог пуст).
- **Каждая итерация = отдельный коммит** на ветке `ralph/auto`. Любую можно ревьюить/откатить:
  `git log --oneline ralph/auto`, `git revert <sha>`.

## Безопасность
- Цикл работает **только на ветке `ralph/auto`**, не на `main`.
- PROMPT запрещает: утечку ключа/сессий/PII, ослабление honesty-гардрейлов DeepSeek, реальные
  отклики и запуск живого браузерного флоу. Верификацию делают Opus-ревьюеры + `npm test`.
- **Про `-Yolo` / `--yolo`:** это `--dangerously-skip-permissions` — агент выполняет любые
  разрешённые инструменты без спроса. Запускай только в этом репозитории, на ветке `ralph/auto`,
  с осознанием рисков. Безопасный режим по умолчанию ограничивает инструменты allowlist'ом и
  авто-применяет только правки файлов.
- Рекомендация: первые 1–2 итерации прогони с маленьким `-MaxIterations 1`, проверь коммит и
  `PROGRESS.md`, потом увеличивай.

## Куда смотреть после прогона
- `ralph/PROGRESS.md` — что сделано/заблокировано.
- `ralph/ROADMAP.md` — сколько задач закрыто (`[x]`).
- `ralph/logs/iter-*.log` — полный вывод итераций.
- `git log --oneline ralph/auto` — коммиты. Когда доволен — мёрж в `main`.
